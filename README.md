# EcoPortal: Secure MERN Authentication System

A professional Full-Stack application developed as part of the MERN Stack Lab course. This project demonstrates responsive UI design, secure JWT-based authentication, and role-based dashboard protection.

## 🛠️ Tech Stack
- **Frontend:** React 19, Material UI (MUI), Axios, React Router Dom
- **Backend:** Node.js, Express.js, JSON Web Tokens (JWT), CORS
- **Deployment:** Vercel (Frontend), Render (Backend)

## 🧪 Experiments Covered
1. **Experiment 1:** Developed a responsive Login UI with client-side validation using React.
2. **Experiment 2:** Implemented JWT Authentication and integrated Frontend with a RESTful Backend.
3. **Experiment 3:** Created a Protected Dashboard accessible only via valid tokens.

## 📌 Note for Evaluators
The backend is hosted on a free cloud tier (Render). If the login process takes 30-40 seconds during the first attempt, it is due to the server spinning up from "sleep mode." Subsequent requests will be near-instant.
